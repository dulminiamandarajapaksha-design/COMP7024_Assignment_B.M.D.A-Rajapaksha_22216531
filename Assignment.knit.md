## Question 1

### Explanation
The objective of carrying out this analysis is to assess the performance of the products by evaluating the total revenue generated by each product. This is achieved by using the orders data set, which contains information regarding each transaction in terms of the quantity of each product sold, the price at which it is being sold, etc. Additionally, the product data set contains descriptive information regarding each product. This information is linked together by matching the StockCode variable in each data set.

Before carrying out the analysis, it is important to clean the data to ensure that it is accurate. This is achieved by dropping rows that contain cancelled orders, as identified by the invoice column starting with the letter "C." Additionally, rows with negative quantities in the orders data set are removed. This is because negative values in the quantity column indicate invalid sales.

Revenue generated by each transaction is calculated by multiplying the quantity of each product by the price at which it is being sold. This is achieved by the formula Revenue = Quantity * Unit Price.

After that, the data is grouped by each product to calculate the total revenue generated by each product. This is achieved by ranking the products from highest to lowest revenue generated. Additionally, the percentage contribution of the top three products to the total revenue is calculated to assess the percentage contribution of each product to the total revenue generated by the organization.

### Code


``` r
# Data Cleaning
orders_clean <- orders %>%
  filter(!str_detect(InvoiceNo, "^C")) %>%   # remove cancelled orders
  filter(Quantity > 0) %>%                   # remove negative quantities
  mutate(StockCode = toupper(StockCode))    # standardize stock codes

products <- products %>%
  mutate(StockCode = toupper(StockCode))

# Join datasets
data <- orders_clean %>%
  left_join(products, by = "StockCode")

# Remove unwanted descriptions
data <- data %>%
  filter(!Description %in% c("Adjust bad debt", "adjustment", "thrown away")) %>%
  filter(!is.na(Description))

# Calculate revenue
data <- data %>%
  mutate(Revenue = Quantity * UnitPrice)

# Revenue by product
product_revenue <- data %>%
  group_by(StockCode, Description) %>%
  summarise(TotalRevenue = sum(Revenue), .groups = "drop") %>%
  arrange(desc(TotalRevenue))

# Top 10 products
top10 <- product_revenue %>%
  slice_max(TotalRevenue, n = 10)

# Bottom 10 products
bottom10 <- product_revenue %>%
  slice_min(TotalRevenue, n = 10)

# Top 3 contribution
top3 <- product_revenue %>%
  slice_max(TotalRevenue, n = 3)

total_revenue <- sum(product_revenue$TotalRevenue)

top3_percentage <- sum(top3$TotalRevenue) / total_revenue * 100

# Display tables
kable(top10, caption = "Top 10 Products by Revenue")
```
